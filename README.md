# House-Prices-predictions

## Задача — спрогнозировать цену продажи каждого дома. Для каждого идентификатора в тестовом наборе необходимо спрогнозировать значение переменной SalePrice. 

## Метрическая система
Результаты оцениваются по среднеквадратичной ошибке (MSLE),  рассчитанной на основе логарифма прогнозируемого значения и логарифма наблюдаемой цены продажи. (Использование логарифмов означает, что ошибки в прогнозировании стоимости дорогих и дешевых домов будут одинаково влиять на результат.)
